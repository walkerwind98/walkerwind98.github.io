Projects

The list below features a list of projects that I have worked on for various classes, research projects, and passion projects.

Thule Clamp Testing Machine

June - August 2020

In order to automate the compression testing of a clamp, I designed a machine that will be used on the factory floor to assure the quality of products before they are sold. The machine uses a motor, which is lowered onto the clamp by the user, to twist the knob on the clamp and close the jaws to a specified compression force. The testing block that is being grabbed by the clamp contains a load cell, that tells the user how much force the clamp is applying.

I designed this machine for my 2020 summer internship with Zephyr Lock. I used SolidWorks to create a 3D model for the machine and solidify the concept for the prototype. I designed the machine to be cost-effective, user-friendly, and safe to use. I fabricated the custom parts for the prototype using various machine shop tools.

Self Playing Violin

June - December 2019

This project, which I began working on in June 2019, is tasked with creating a violin that can be controlled autonomously. In developing this working prototype that can be later carried to market, I have learned about troubleshooting, designing a system, and rapid prototyping. The majority of my contribution has been to developing the controls and electronics to act in concert with the physical motion of the violin bow.

Brass Acid Bleed Project

June - August 2019

As part of a Mechanical Engineering Internship at the Tufts Center for Engineering Education and Outreach, I conducted a scientific study to determine the origin of a defect that affects the manufacturing of trombones at a major instrument manufacturer.  In order to understand what was causing the reaction, I designed tests to eliminate variables in the reaction, gathered information about the factory conditions and processes, and documented my theories and results.

 

In order to conduct research on brass samples, I constructed my own humidity chamber. This design integrated humidity sensors with a fan system that allows for the user to regulate the humidity at which materials are tested. The casing and hinge was designed in SolidWorks, and cut using a laser cutter. The electro-mechanical system is powered using a Raspberry Pi, using Python libraries to create a Flask server that hosts a local website to display statistics about the temperature and humidity.

Lego Wall-E

January - May 2020

Under the guidance of Professor Chris Rogers, a friend and I designed and built a small robot resembling Pixar's Wall-E robot that was piloted using a chest-mounted controller suit. The controller used gyroscopic sensors to control the motion of the robot. I coded the robot using the Lego EV3 microPython libraries.

Cornhole Data Analysis

March - May 2020

For my Instruments and Experimentation class, I used video processing software and data analysis to analyze my ability to play cornhole. My group and I studied all of the measurable variables that went into a cornhole throw, the throwing performance over time, and the error that was generated by the video analysis software. To investigate all of these topics, I used a sports video analysis software, Kinovea, to generate metrics on each cornhole throw. I used Matlab code to process the data collected on each of the cornhole throws and provide quanititative results on each of the topics we studied.
